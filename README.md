### Deployment
**1. ec2 í™˜ê²½ êµ¬ì„±**
>  Install dependencies
> ```
> sudo apt update
> sudo apt install -y git openjdk-17-jdk
> ```

> project clone & build
> ```
> git clone https://github.com/CrossNoteSWU/BE.git
> ./gradlew clean build -x test
> ```
**2. Docker & Redis**
> Install Docker 
> ```
> curl -fsSL https://get.docker.com -o get-docker.sh
> sudo sh get-docker.sh
> ```

> Run Redis container
> ```
> docker pull redis:latest
> docker run -d \
>   --name redis-crossnote \
>   -p 6379:6379 \
>   redis
> ```

**3. Spring Boot ì‹¤í–‰ ìœ ì§€**
> ```
> sudo systemctl restart crossnote
> ```

> ```

**5. Nginx + HTTPS**
> Configure Nginx site
> ```
> sudo nano /etc/nginx/site-available/https://cross-note.com
> ```
> Reload nginx
> ```
> sudo nginx -t
> sudo systemctl reload nginx
> ```
